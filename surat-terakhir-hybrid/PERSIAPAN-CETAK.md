# PERSIAPAN CETAK — Surat Terakhir Hybrid

> Panduan mencetak kartu fragmen untuk mode hybrid.

---

## 1. Bahan yang Dibutuhkan

| Bahan | Jumlah | Keterangan |
|---|---|---|
| Kertas A4 | 2-3 lembar per ronde | Untuk cetak kartu fragmen |
| Gunting | 1 | Untuk potong kartu |
| Amplop (opsional) | 6-8 per ronde | Untuk tutup kartu (misteri) |
| Pensil/pulpen | 1 per pemain | Untuk voting |

---

## 2. Cara Cetak

### Step 1: Buka cetak.html
- Buka file `cetak.html` di browser (Chrome/Firefox)
- Tidak perlu internet, buka langsung dari file

### Step 2: Pilih Ronde
- Pilih ronde yang mau dicetak (1-5)
- Setiap ronde punya kombinasi kitab berbeda

### Step 3: Print
- Klik tombol **"Print"** atau `Ctrl+P`
- Setting: **A4**, **Portrait**, **Tanpa margin** (atau margin minimal)
- Pastikan **"Background graphics"** centang (untuk warna)

### Step 4: Potong Kartu
- Gunting sesuai garis putus-putus
- Setiap kartu berisi:
  - Nomor kartu (untuk identifikasi)
  - Fragmen ayat (3-15 kata)
  - **Tidak ada nama kitab** (ini rahasia!)

---

## 5. Tips Cetak

- **Cetak lebih banyak** — siapkan 8 kartu per ronde (untuk fleksibilitas)
- **Kertas tebal** — gunakan kertas 80gsm+ supaya kartu tidak tembus pandang
- **Warna berbeda** (opsional) — cetak di kertas berbeda per kitab untuk memudahkan sorting
- **Laminasi** (opsional) — kalau mau dipakai ulang, laminasi kartu
- **Amplop** — masukkan kartu ke amplop tertutup supaya pemain tidak lihat sebelumnya

---

## 6. Setup Hari H

### Sebelum Acara:
1. Cetak semua kartu untuk 5 ronde
2. Potong dan sortir per ronde
3. Masukkan ke amplop (opsional): label "Ronde 1", "Ronde 2", dst
4. Siapkan kertas voting (bisa kertas biasa, potong kecil)

### Saat Acara:
1. Moderator buka `index.html` di HP/laptop
2. Input nama pemain
3. Klik "Mulai Ronde 1"
4. Dashboard tampilkan distribusi (siapa dapat apa)
5. Moderator bagikan kartu sesuai distribusi
6. Jalankan game!

---

## 7. Troubleshooting

| Masalah | Solusi |
|---|---|
| Kartu tidak muncul di print | Pastikan "Background graphics" centang di print settings |
| Teks terpotong | Perbesar margin atau kurangi font size |
| Kartu terlalu kecil | Gunakan landscape atau kurangi jumlah kolom |
| Tidak ada printer | Tulis fragmen manual di kertas kecil |
