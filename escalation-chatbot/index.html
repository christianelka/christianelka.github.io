Koreksi Hashtag

Panduan Tester EscBot

Hai! Terima kasih sudah bantu nge-tes EscBot. Tujuan tes ini sederhana: cek apakah hashtag eskalasi yang muncul sudah sesuai dengan konteks yang Anda ketik. Kalau belum sesuai, Anda tinggal koreksi. Hasil koreksi nanti dipakai untuk memperbaiki database hashtag.

AI Mode sudah aktif: Anda bisa pakai pencarian lokal (default) ATAU aktifkan AI Mode di sidebar untuk hasil yang lebih akurat. Pengaturan AI sudah dikonfigurasi - tinggal aktifkan toggle AI Mode di sidebar.

Cara pakai

  1. 1
    Ketik notes / konteks di kolom chat di bagian bawah, lalu tekan Enter.
    Contoh: "kendala tagihan MEC bookpayment", "aktivasi paket flash corporate", atau notes asli dari case yang Anda tangani.
    Tekan Shift + Enter kalau mau ganti baris tanpa kirim.
  2. 2
    Bot akan menampilkan sampai 5 opsi hashtag hasil pencocokan. Setiap opsi berisi nama Group, KIP, hashtag, dan skor kecocokan (Tinggi / Sedang / Rendah).
    Klik kotak hashtag berwarna ungu untuk copy ke clipboard.
  3. 3
    Tentukan apakah opsi tersebut benar atau salah via 3 tombol di bawah tiap kartu:
    Benar, pakai ini ->  Klik kalau hashtag sudah benar. Bot akan ingat preferensi ini.
    Group salah, koreksi ->  Klik kalau Group / hashtag tidak cocok. Modal koreksi terbuka, pilih hashtag yang benar dari list.
    Diskusi dgn AI ->  (Hanya saat AI Mode aktif) Diskusi natural language - ketik koreksi seperti "bukan itu, harusnya group MEA", AI akan revisi pilihan.
  4. 4
    Saat melakukan koreksi (modal):
    • Cari hashtag yang benar di kolom pencarian (ketik nama Group, KIP, atau kata kunci).
    • Klik salah satu opsi pada list - opsi terpilih akan ter-highlight.
    • Tekan tombol Simpan Koreksi.
    Kalau tidak ada satupun dari 5 opsi yang cocok, tetap pilih opsi terdekat lalu lakukan koreksi - jangan dibiarkan saja.
  5. 5
    Riwayat Chat tersimpan otomatis di sidebar.
    Tiap query yang Anda kirim membuat thread baru. Klik thread di sidebar untuk lihat percakapan lama. Tombol Chat Baru untuk mulai thread fresh.
Penting: Mohon dikoreksi setiap kali ada hashtag / group eskalasi yang menurut Anda tidak sesuai. Inilah data utama yang saya butuhkan untuk memperbaiki database.

Pembelajaran & Export

Setiap kali Anda menekan "Benar, pakai ini", menyimpan koreksi, atau diskusi dengan AI, datanya tersimpan otomatis di sidebar bagian Pembelajaran -> Koreksi tersimpan. Counter di sana akan bertambah.

Banner kuning di atas chat akan muncul kalau Anda punya data belum di-export. Klik tombol export di banner itu, atau lewat sidebar manual.

Sebelum menutup tab / selesai testing, mohon lakukan ini:
  1. Buka sidebar (klik ikon menu kalau di mobile).
  2. Di kartu Pembelajaran, klik tombol Export untuk download koreksi.
  3. Di kartu Riwayat Chat, klik tombol Export untuk download riwayat percakapan (opsional tapi membantu kalau ada bug).
  4. File JSON akan terunduh otomatis (nama: escbot-corrections-YYYY-MM-DD.json dan escbot-history-YYYY-MM-DD.json).
  5. Kirimkan file koreksi ke saya agar saya bisa update databasenya.
Jangan klik Reset / Bersihkan di kartu Pembelajaran atau Riwayat sebelum file di-export - tombol itu akan menghapus data secara permanen.
